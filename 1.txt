## 📌 Project Description
This is a simple beginner-level web development project created using HTML and CSS.  
It demonstrates basic webpage structure, styling, tables, lists, and formatting.

The purpose of this project is to practice fundamental web development skills.

🚀 Features

- Basic HTML page structure  
- Headings from H1 to H6  
- Ordered lists and definition lists  
- Student data tables  
- Equipment information table  
- External CSS styling  
- Hover effect on links  
- Centered layout elements
